# Simple Python Calculator

A beginner-friendly Python calculator that supports addition, subtraction, multiplication, and division.  
This project demonstrates basic arithmetic operations along with **exception handling** for invalid inputs and division by zero.

## Features

- Addition (+), Subtraction (-), Multiplication (*), Division (/)
- Handles invalid inputs using `try/except`
- Prevents division by zero
- Interactive loop allows multiple calculations until the user chooses to exit
